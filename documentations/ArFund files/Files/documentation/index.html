Installation
Application Installation Process

Installing our Laravel ArFund project is simple. You can set it up on your cPanel with ease, following the default straightforward process. No coding knowledge is needed; just a few easy steps, and you're ready to go. We've designed the installation process to be seamless and user-friendly.

Steps to Set Up:

  • 1. Upload the Project Files

  • Simply upload the entire downloaded file to your server after purchasing.

  • 2. Extract the Files

  • Extract or unzip the file. Then, move all the files from the Files folder into your desired installation directory. Be sure to include both the index.php and .htaccess files during this process.

  • 3. Import the Database

  • Go to your hosting control panel, navigate to the database section, and create a new MySQL database.

  • Navigate to the database you created for your project.

  • Look for the "Import" option in phpMyAdmin and click on it. Then, click on the "Choose File" button to select the SQL file you received.

  • Once you've selected the SQL file, click on the "Go" or "Import" button to execute the import process. Depending on the size of the SQL file, this may take some time.

  • After the import process is complete, you should see a success message confirming that the SQL file has been imported successfully into your database.

  • 4. Configure the .env File

  • Open the .env (base/.env) file and update the following lines with your database credentials

  • 5. System is ready now

  • Now please browse http://your-site-url. Your system should be ready to use.

If you're still having trouble installing the system, please reach out to us. We offer free installation for cPanel-based hosting.

System Configuration
System Configurations overview

You can adjust the system settings of your website in the system configurations.

  • User Registration : You can turn the sign-up feature on or off.

  • Force SSL : If you turn this on, it will make sure all traffic uses HTTPS.

  • Secure Password : If you turn this on, users must set a secure password when they sign up or change their password.

  • Agree Policy : If you turn this on, users must agree to the system policies during the sign-up process.

  • KYC Verification : If you turn this on, users must submit their KYC details after signing up.

  • Email Verification : You can choose to turn the Email Verification system on or off.

  • Email Notification : You can decide whether to turn on or off the Email Notification system.

  • Mobile Verification : You can decide whether to turn on or off the Mobile Verification system.

  • SMS Notification : You can choose whether to turn on or off the SMS Notification system.

  • Language Option : If you turn off this module, the language option won't be visible on the system.
